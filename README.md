# Landscape 3D

A small React + Three.js experience that blends a cinematic landing page with realtime scenes and a separate solar system route. The UI follows an atomic design structure and component-scoped BEM styling.

## Tech Stack
- React 18
- TypeScript
- Vite
- React Router
- three.js
- @react-three/fiber

## Getting Started

### Install
```
pnpm install
```

### Development
```
pnpm dev
```

### Production Build
```
pnpm run build
```

### Preview Production Build
```
pnpm run preview
```
